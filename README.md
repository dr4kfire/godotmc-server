# GodotMC - A Minecraft Java edition server
This is an implementation of the minecraft java server made completly in godot. The main focus is to allow easier ways for creating minecraft plugins, servers and experiences. NO AI was used in creation of this project.

### 📦 FEATURES:
These are the things that are implemented right now
- none

### 🎯 ROADMAP:
This is a (somewhat chronological) list of the next features
- SLS and PING protocols
- LOGIN protocol

Please keep in mind that this is a hobby project that I work on in spare time, so don't expect much from it if anything (feel free to contribute though!)
